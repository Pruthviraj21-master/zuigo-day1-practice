# Cost Estimate

## Example Request

Input Tokens: ~300

Output Tokens: ~150

Total Tokens: ~450

---

## Estimated Cost

The exact cost depends on the selected Gemini model.

For this prototype, each request is expected to cost only a fraction of a cent.

Estimated cost for 1,000 users making one request each is typically well under one US dollar when using a lightweight Gemini model.

---

## Cost Optimization

- Keep prompts short.
- Retrieve only the most relevant document.
- Limit response length.
- Cache repeated questions when possible.
