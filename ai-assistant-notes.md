
# What I learned about AI coding assistants — Day 1

## Good at:
- Explaining unfamiliar code quickly
- Spotting syntax errors and explaining them in plain language
- Writing boilerplate/small scripts fast

## Where it fails / needs supervision:
- Doesn't know my specific project context unless I give it
- Can generate code that "looks right" but has subtle bugs
- I still need to test and understand everything before shipping it
