# Day 5 — Design Notes (CreatorBridge)

## Note

Wireframes were not created in Figma for this pass. Since CreatorBridge
already has a working MERN-stack frontend, these are screenshots of the
actual built screens, used as the design reference for Day 5. Figma
wireframes can be added later if required.

## Screens & Traceability

### 1. Landing Page (`screenshots/01-landing.png`)

- Sections: Nav bar, announcement banner, hero headline, CTA buttons
  (I'm a Creator / I'm a Business), live collaboration preview cards
- Maps to: [user story — e.g. "As a visitor, I want to understand what
  CreatorBridge does and choose my role"]

### 2. Post Collaboration + Requests (`screenshots/02-post-collaboration.png`)

- Sections: Post form (title, description, category, budget, platform,
  location), list of posted requests with applicant details
- Maps to: [user story — "As a business, I want to post a collaboration
  request and review applicants"]

### 3. Business Dashboard (`screenshots/03-business-dashboard.png`)

- Sections: Account details, stats (Requests/Applicants/Open), posted
  requests list with "Review Applicants" action
- Maps to: [user story — "As a business, I want a summary view of my
  activity"]

### 4. Collaboration Discovery (`screenshots/04-collaboration-discovery.png`)

- Sections: Filters (Industry, Budget, Status, Platform, Location),
  results list
- Maps to: [user story — "As a creator, I want to search/filter open
  collaborations"]

### 5. Creator Dashboard (`screenshots/05-creator-dashboard.png`)

- Sections: Account details, stats (Applications/Pending/Contact),
  applications list
- Maps to: [user story — "As a creator, I want to track my applications"]
